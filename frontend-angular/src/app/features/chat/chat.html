Sobre este Proyecto

help_outline ¿Cómo usar Emma?

Emma te asistirá simulando el proceso de compra a través de 5 tiendas integradas. Puedes explorar categorías, consultar precios y armar tu carrito.

  • 👉 "menú": Repite las opciones de la tienda actual.
  • 👉 "agregar al carrito": Añade el producto que estás viendo.
  • 👉 "salir": Regresa al menú principal en cualquier momento.

code Stack Tecnológico

Desarrollado aplicando principios de Clean Architecture, asegurando la calidad técnica mediante un núcleo inmutable y funciones puras.

Frontend

Angular, TypeScript, Tailwind CSS. UI/UX interactiva con integración de reconocimiento de voz (Whisper API) y habla (Edge_tts).

Backend & Algoritmos

Motor de búsqueda semántica utilizando Distancia de Levenshtein y Similitud del Coseno. Navegación estructurada del catálogo implementada mediante algoritmos de grafos (BFS y DFS).
